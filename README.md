# Dossier-Gratitude
Dossier: Gratitude — sebuah aplikasi web jurnal syukur harian dengan desain neo-brutalist/classified dossier. Pengguna bisa menulis catatan syukur harian, menyimpannya di browser, melihat tanggal yang sudah terisi lewat kalender, dan membuka arsip catatan sebelumnya dalam tampilan recap.
