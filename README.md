
# Dossier: Gratitude

Dossier: Gratitude adalah aplikasi web jurnal syukur harian dengan gaya visual neo-brutalist dan tema dossier/classified document. Aplikasi ini dibuat untuk membantu pengguna mencatat hal-hal yang disyukuri setiap hari secara sederhana, cepat, dan tetap menarik secara visual.

## Fitur

- Menulis jurnal syukur harian
- Menyimpan data langsung di browser menggunakan `localStorage`
- Kalender untuk melihat tanggal yang sudah memiliki catatan
- Arsip/rekap semua jurnal yang pernah disimpan
- Kutipan harian yang berubah secara acak
- Shortcut keyboard untuk navigasi lebih cepat
- Desain responsive untuk desktop dan mobile

## Tampilan Utama

Aplikasi memiliki 3 bagian utama:

### 1. Jurnal
Halaman utama untuk menulis dan menyimpan catatan syukur hari ini.

### 2. Kalender
Menampilkan kalender bulanan dan menandai tanggal yang sudah memiliki entri jurnal.

### 3. Arsip
Menampilkan daftar semua catatan yang sudah disimpan, diurutkan dari yang terbaru.

## Teknologi yang Digunakan

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Tailwind CSS (via CDN)
- Google Fonts
- Font Awesome
